# Discord Bot

A simple Discord bot example for handling commands and events.

## Features
- Responds to commands
- Easy to configure
- Minimal, extensible codebase

## Requirements
- Node.js 16+ and npm
- A Discord bot token

## Installation
1. Clone the repo:
    git clone <repo-url>
2. Install dependencies:
    npm install

## Configuration
Create a .env file in the project root:
```
DISCORD_TOKEN=your-bot-token
PREFIX=!
```

## Usage
Start the bot:
```
npm start
# or
node index.js
```
