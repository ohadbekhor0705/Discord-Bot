# Discord Bot

A simple Discord bot example for handling commands and events.

## Features
- Responds to commands
- Easy to configure
- Minimal, extensible codebase

## Requirements
- A Discord bot token

## Installation
Clone the repo:
    git clone <repo-url>
install

## Configuration
Create a .env file in the project root:
```
DISCORD_TOKEN=your-bot-token
PREFIX=!
```

## Usage
Start the bot:
```
python main.py
```
